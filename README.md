# ShopnStitch (E-commrece App)

# Introduction
  An e-commrece app to buy stitched and un-stitched cloths ...

## Technologies
Project is created with 
* Next.js, React, Hooks, 
* Redux 
* Tailwind Elements,
* GraphCMS. 

## Setup:
$ run `npm i && npm run dev || yarn i && yarn run dev` for starting the application.
